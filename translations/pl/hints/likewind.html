Nie mam pojęcia jak rozwiązać to zadanie.

Może Pan znaleźć pierwsze wystąpienie szukanego znaku w ciągu znaków. Następnie proszę przesunąć się o jeden znak dalej i ponownie znaleźć ten sam znak. Będzie to jego drugie wystąpienie w ciągu.

Od czego zacząć?

Przede wszystkim należy upewnić się, że wymagany znak rzeczywiście występuje w ciągu co najmniej dwa razy. Jeśli tak nie jest, program powinien zwrócić None. Pomoże Państwu w tym metoda str.count().

Jak powinien wyglądać kod i jak działa?

Pierwsza część programu może wyglądać następująco:

def second_index(text, symbol):

    if text.count(symbol) < 2: 
        return None

Na przykład, jeśli mamy ciąg "jakiś tekst" i sprawdzamy literę "e", to program nie zwróci None. Ale jeśli zastąpimy literę 's' za argument symbolu, wówczas program zwróci None, ponieważ 'some text'.count ('s') == 1 a warunek 1 < 2 jest spełniony.

Co dalej?

Teraz, gdy upewnili się Państwo, że w ciągu znaków rzeczywiście znajdują się co najmniej 2 wymagane znaki, nadszedł czas na znalezienie pozycję pierwszego wystąpienia. Metoda find pomoże Państwu w tym. Zwraca ona indeks pierwszego wystąpienia znaku znalezionego w ciągu. Na przykład:

text = 'abcdec'
symbol = 'c'
first = text.find(symbol, 0)

Proszę zauważyć, że 0 jest opcjonalne w nawiasach. To tylko pokazuje, gdzie funkcja zaczyna szukać symbolu (0 oznacza, że zaczyna od samego początku ciągu). W tym przypadku "pierwszy" będzie równy 2, ponieważ znak "c" znajduje się w pozycji o indeksie 2.

W kodzie będzie to wyglądać następująco ...

Coś takiego:

def second_index(text: str, symbol: str) -> int | None:
            
    if text.count(symbol) < 2: 
        return None
    first = text.find(symbol)

Tak więc, jeśli w naszym tekście wejściowym wymagany element występuje mniej niż 2 razy, funkcja zwróci None. Jeśli wystąpi 2 lub więcej razy - funkcja znajdzie pierwsze wystąpienie znaku w ciągu znaków.

Ale my potrzebujemy drugiego wystąpienia, prawda?

Zgadza się. Dlatego rozpoczynamy wyszukiwanie nie od samego początku ciągu, ale od znaku następującego po pozycji który uznaliśmy za "pierwszy". Aby to osiągnąć, wystarczy zmienić pozycję początkową wyszukiwania z 0 na pierwszą+1.

Potrzebuję więcej pomocy!

Ostateczna wersja programu będzie wyglądać następująco:

def second_index(text: str, symbol: str) -> int | None:
            
    if text.count(symbol) < 2: 
        return None
    first = text.find(symbol)
    return text.find(symbol, first + 1)

Jeśli spróbujemy znaleźć znak "a" w zdaniu "Uczę się Pythona i niczego więcej" za pomocą tej funkcji, otrzymamy 20. Ponieważ najpierw ustalimy, że znak 'a' występuje po raz pierwszy w pozycji o indeksie 6 (first = 6), a następnie, gdy zaczniemy szukać od następnego znaku (first + 1), ustalimy, że następny znak "a" występuje w pozycji o indeksie 20.
