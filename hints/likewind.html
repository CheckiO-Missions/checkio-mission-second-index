You can find the first occurrence of the character you are looking for in a string. After that move one character further along the string and find the same character again. This will be its second occurrence in the string.

Where do I start?

First of all, you need to make sure that the required character actually occurs in the string at least two times. If it's not the case, the program should return None. The str.count() method will help you.

What should code look like and how does it work?

The first part of the program might look like this:

def second_index(text, symbol):

    if text.count(symbol) < 2: 
        return None

For example, if we have a string 'some text' and we are checking the letter 'e', then the program won't return None. But if we substitute the letter 's' for the symbol argument, then the program will return None, since 'some text'.count ('s') == 1 and the condition 1 < 2 is met.

What’s next?

Now, when you've made sure that there really are at least 2 required characters in the string, it's high time to find the position of the first occurrence. The find method will help you with that. It returns the index of the first character occurrence found in the string. For example:

text = 'abcdec'
symbol = 'c'
first = text.find(symbol, 0)

Please note, that 0 is optional in parentheses. This only shows where the function starts looking for symbol (0 means that it starts from the very beginning of the string). In this case, the 'first' will equal 2, since the character 'c' is in the position with the index 2.

In code it will look like ...

Something like this:

def second_index(text: str, symbol: str) -> int | None:
    if text.count(symbol) < 2: 
        return None
    first = text.find(symbol)

Thus, if in our input text the required element occurs less than 2 times, the function will return None. If it occurs 2 or more times - the function will find the first occurrence of the character in a string.

But we need the second occurrence, don’t we?

That's right. Therefore, we are starting the search not from the very beginning of the string, but from the character following the position that we've found as the 'first'. To accomplish this, we just need to change the starting position of the search from 0 to the first+1.

The final version of the program will look like this:

def second_index(text: str, symbol: str) -> [int, None]:
    if text.count(symbol) < 2: 
        return None
    first = text.find(symbol)
    return text.find(symbol, first + 1)

If we try to find the character 'a' in the sentence "I'm learning Python and nothing else" by using this function, we'll get 20. Because at first we will determine that the character 'a' occurs the first time in the position with the index 6 (first = 6), and then, when we start looking from the next character (first + 1), we'll determine that the next character 'a' occurs in the position with the index 20.
