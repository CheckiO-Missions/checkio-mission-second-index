您可以在字符串中找到第一个出现的字符。然后沿字符串向前移动一个字符，再次找到相同的字符。 这将是它在字符串中的第二次出现。

从哪里开始？

首先，您需要确保所需字符在字符串中至少出现两次。 如果没有，程序将返回None。 str.count() 方法将帮助您。

下一步是什么？

现在，当您确定字符串中确实至少有2个必填字符时，是时候找到 第一个出现字符的位置。find 方法 方法可以帮助您。 它返回字符串中第一个出现的字符的索引。例如

text = 'abcdec'
symbol = 'c'
first = text.find(symbol, 0)

请注意，括号中的0是可选项。这只显示了函数 开始查找符号的位置（0表示从字符串的最开头开始查找）。 在本例中，"first "等于2，因为字符 "c "位于索引为2的位置。

def second_index(text: str, symbol: str) -> int | None:
            
    if text.count(symbol) < 2: 
        return None
    first = text.find(symbol)
