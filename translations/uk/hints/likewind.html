Ти можеш знайти перше входження символу, який шукаєш, у рядку. Після цього перемістись на один символ далі по рядку і знову знайди той самий символ. Це буде його друге входження в рядок.
З чого мені почати?
Перш за все, тобі потрібно переконатися, що потрібний символ дійсно зустрічається в рядку принаймні двічі. Якщо це не так, програма має повернути None. Тобі допоможе метод str.count().
Як має виглядати код і як він працює?
Перша частина програми може виглядати так: 
def second_index(text, symbol):
    if text.count(symbol) < 2:
        return None
Наприклад, якщо у нас є рядок "some text" і ми перевіряємо букву «e», то програма не поверне None. Але якщо ми оберемо літеру 's' для пошуку, тоді програма поверне None, оскільки 'some text'.count ('s') == 1 і виконується умова 1 < 2.
Що далі?
Тепер, коли ти знаєш, що в рядку дійсно принаймні два обов'язкових символи, настав час знайти позицію першого входження. Метод str.find() допоможе тобі у цьому. Він повертає індекс першого символу, знайденого в рядку. Наприклад:
text = 'abcdec'
symbol = 'c'
first = text.find(symbol, 0)
Зверни увагу, що 0 необов’язковий у дужках. Це лише показує, звідки функція починає пошук символу (0 означає, що він починається з самого початку рядка). У цьому випадку "first" матиме індекс "2", оскільки символ "с" стоїть на позиції з індексом 2.
Як у коді це виглядатиме?
Щось на зразок цього: 
def second_index(text: str, symbol: str) -> [int, None]:
    if text.count(symbol) < 2:
        return None
    first = text.find(symbol)
Таким чином, якщо у нашому вхідному тексті необхідний елемент зустрічається менше ніж двічі, функція поверне None. Якщо він зустрічається 2 або більше разів - функція знайде перше входження символу в рядку.
Але нам потрібен другий випадок, чи не так?
Це вірно. Тому ми починаємо пошук не з першого символу у рядку, а з наступного за позицією символу. Щоб досягти цього, нам просто потрібно змінити початкову позицію пошуку з 0 на first+1.
Мені потрібна допомога!
Остаточна версія програми буде виглядати так: 
def second_index(text: str, symbol: str) -> [int, None]:
    if text.count(symbol) < 2:
        return None
    first = text.find(symbol)
    return text.find(symbol, first + 1)

Тому що спочатку ми визначимо, що символ "і" зустрічається вперше в позиції з індексом 16 (first = 16), а потім, коли ми починаємо шукати з наступного символу (first + 1), ми визначимо, що зустрічається наступний символ "і" в позиції з індексом 19.
